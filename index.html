Proyectos

Sistema de registro

Sistema administrativo de registro de usuarios, es un sistema donde se registran los usuarios con diferentes roles, este sistema esta desarrollado con las tecnologías: Vuejs 3, Vuetify 3,Axios, NodeJS ExpressJS etc..

Página web MDPyEP

Página web de Ministerio de Desarrollo Productivo y Economía Plural tiene como objetivo el poder publicar noticias o novedades que se desarrollan por parte de sus autoridades, para que el público en general tengan la información actualizada.

Monitoreo de Metas

Página web de Monitoreo de Metas SPCC 2024 Bolivia tiene como objetivo el poder publicar las metas y certificaciones cumplidas por cada departamento de Bolivia.

Aplicación Control de Gastos

Control de Gastos la aplicación que te ayuda a llevar un registro de tus gastos y administrar tus finanzas personales, desarrollado con Flutter y Dart.

Blog Tecno Evolucíon

Sistema Marketplace

Sistema Marketplace es un sistema de alta gama, es una tienda de tiendas llamadas a la acción, es decir los usuarios pueden registrar sus tiendas y también sus productos y poder categorizarlas.
