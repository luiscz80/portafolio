Frontend

Sistema de Registro de Usuarios Este sistema permite a los usuarios registrarse de manera rápida y segura utilizando Vue.js 3, Vuetify 3, Axios y Node.js con Express.js. Características Principales Interfaz Amigable: Vuetify 3 proporciona una UI moderna y responsiva. Registro Eficiente: Formulario con validaciones en tiempo real para asegurar datos correctos. Manejo de Errores: Axios gestiona respuestas del servidor, mostrando mensajes claros ante errores. Seguridad: El backend en Node.js utiliza encriptación de contraseñas y validación de datos. Autenticación: Uso de JWT para acceder a áreas protegidas de la aplicación.

Inicio de sesión

Se muestra el formulario login para el inicio de sesión.

Usuarios registrados

Se muestra en datatable la información de todos los usuarios registrados en el sistema.

Perfil de usuario

Se muestra la información del perfil de usuario que inició la sesión.

Actualizar Perfil y Seguridad del usuario, notificaciones, preferencia de tema

Se muestra los datos del usuario autenticado, con la opción de cambiar nueva contraseña, subir imagen (avatar), activar notificaciones, preferencias de tema (light, dark).

Roles registrados

Se muestra en datatable la información de todos los roles registrados en el sistema.

Permisos registrados

Se muestra en datatable la información de todos los permisos registrados en el sistema.

Gráficos estadísticos

Se muestra en gráfico de vue-chartjs total de usuarios por rol en el sistema.

Exports

Se muestra un select para elegir el formato de exportar Datos de Usuarios en: PDF, EXCEL y CSV.

Logs del Sistema

Se muestra información de logs del sistema con opción de exportar, buscar por palabras claves y filtrar por fechas.

Backend

Documentación de la API: Registro de Usuarios
Versión: 1.0.0

Descripción:
La API de Registro de Usuarios proporciona una interfaz robusta y segura para gestionar la creación y administración de usuarios en aplicaciones web y móviles. Esta API está diseñada para soportar operaciones CRUD (Crear, Leer, Actualizar, Eliminar) y se basa en estándares de seguridad modernos mediante el uso de JSON Web Tokens (JWT) para la autenticación.

Características Principales:

  • Operaciones CRUD: Permite la gestión completa de usuarios, incluyendo el registro, actualización de datos, obtención de información y eliminación de cuentas.
  • Autenticación Segura: Implementa JWT para garantizar que solo los usuarios autenticados puedan acceder a los recursos protegidos de la API.
  • Validación de Datos: Utiliza Joi para la validación de la entrada de datos, asegurando que la información proporcionada cumple con los requisitos establecidos, lo que reduce el riesgo de errores y vulnerabilidades.

Endpoints:
La API incluye los siguientes endpoints:
  • Registro de Usuario: Permite la creación de nuevos usuarios mediante un proceso de registro.
  • Inicio de Sesión: Proporciona un mecanismo para que los usuarios existentes inicien sesión de manera segura.
  • Gestión de Usuarios: Incluye funcionalidades para obtener, actualizar y eliminar usuarios.

Uso:
La documentación detallada de cada endpoint, junto con ejemplos de solicitudes y respuestas, está disponible para facilitar la integración y uso de la API en diferentes entornos.
